# Leaf & Light Studio Website

Static one-page website for **Leaf & Light Studio**.

## What is included

- `index.html` — main website
- `styles.css` — full responsive design
- `script.js` — mobile menu, portfolio filters, copy email, small reveal animations
- `assets/` — placeholder SVG visuals

## How to preview locally

Open `index.html` in your browser.

## How to publish on GitHub Pages

1. Create a new GitHub repository, for example `leaf-light-studio-site`.
2. Upload all files from this folder to the repository root.
3. Go to **Settings → Pages**.
4. Set source to **Deploy from a branch**.
5. Choose branch `main` and folder `/root`.
6. Save. GitHub will generate the public website URL.
