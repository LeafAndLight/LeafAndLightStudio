# Content checklist for final version

## Brand
- Final logo PNG/SVG
- Brand colors
- Preferred studio tagline

## Games
For each game:
- Name
- Status
- Platform
- Store link
- 1-line pitch
- 3-5 features
- 5-10 clean screenshots
- Trailer or GIF

## Portfolio
For each project:
- Project name
- Can it be shown publicly? yes/no
- Can the game/client name be shown? yes/no
- Your exact contribution
- Category: VR game, environment art, prototype, collaboration, etc.

## Contact
- Email
- LinkedIn
- Meta profile
- ArtStation / Sketchfab / YouTube if available
